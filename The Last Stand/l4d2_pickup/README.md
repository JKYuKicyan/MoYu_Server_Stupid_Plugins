# Pick Up Changes

### Introduction
- A fixed version that correctly block switching to dual pistols when picking up another pistol.

<hr>

### Installation
1. Put the **l4d2_pickup.smx** to your _plugins_ folder.
2. Put the **l4d2_pickup.txt** to your _gamedata_ folder.
